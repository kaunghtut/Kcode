🔗 Google Sheets Integration Setup

  1. Create a Google Sheet with columns: ID, Date, Customer, Amount, Status, Description
  2. Make the sheet publicly viewable (Share → Anyone with link can view)
  3. Copy the sheet ID from the URL (between /d/ and /edit)
  4. Paste the Sheet ID below and click "Connect to Google Sheets"
  5. For writing back to sheets, you'll need Google Sheets API setup with service account
